#Tipos de Dados

Conforme o ECMAScript satandard temos 9 tipos de dados:

- Data Types
  - Primitive / Primitive value
  - Structural
  - Structural Primitive

## Primitivos

Primitivos não sofre alterações:

- String
- Number
- Boolean
- undefined
- Symbol
- BigInt

## Estruturais

- Object
  - Array
  * Map
  * Set
  * Date
  * ...

* Function

## Primitivo Estrutural / Structural Root Primitive

- null
